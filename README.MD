为解决目前遥感云计算平台众多，函数与数据资源难以寻找的问题，我们构建一个遥感云计算学习、讨论、交流的平台。

官方网址为: [www.piegee.com](www.piegee.com)

该仓库为源代码文件，安装与启动教程如下：  
  
（1）安装环境

```
npm install
#or
yarn install
```

（2）启动环境

```
npm dev
# or
yarn dev
```

（3）部署环境

```
npm build
# or
yarn build
```
