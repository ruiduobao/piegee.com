## UI

平台提供丰富的交互式UI组件，UI组件的出现为专业用户提供了一个可以直接使用、分享以及探讨遥感技术的方式，为广大非专业的用户降低了应用遥感数据和算法的门槛。

UI组件包括文本、表格、单选框、输入框、颜色选择器、日期选择器、数字输入框、复选框、开关按钮、滑动条、下拉列表、按钮等。
